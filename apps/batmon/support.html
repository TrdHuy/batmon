Basic Troubleshooting

Controller is not detected

  • Confirm the controller is paired and connected in Android Bluetooth settings.
  • Disconnect and reconnect the controller, then reopen Batmon / Glimpse.
  • Grant Bluetooth permission if Android asks for it.

Battery information is unavailable

  • Some controllers or Android versions may not expose battery data.
  • Try reconnecting the controller or checking with another compatible controller.

Overlay is not visible

  • Grant Android overlay permission for Batmon / Glimpse.
  • Enable Live Battery Overlay inside the app.
  • Check whether the system or device vendor restricts overlay display.

Monitoring stops unexpectedly

  • Allow notifications if Android requires notification permission.
  • Review battery optimization settings if the device aggressively stops background work.

Report a Problem

In the app, open Report Problem to prepare an email report. The email client opens before anything is sent, so the user can review or edit the report first.

Helpful reports include the controller model, Android device model, Android version, what happened, and steps to reproduce the issue.
