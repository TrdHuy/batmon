Privacy Policy

This policy explains how Batmon / Glimpse handles information when used to monitor Bluetooth game controllers and display controller status.

Data Collection

Batmon / Glimpse is designed to run locally on the user's Android device. The app does not automatically collect, upload, sell, or share personal data with Synclab or third parties.

The app does not collect sensitive personal information such as contacts, messages, photos, precise location, account credentials, or payment data.

Bluetooth and Device Information

The app uses Bluetooth and Android input device APIs to detect connected game controllers and display available controller information. This may include controller name, connection state, device identifiers exposed by Android, vendor/product identifiers, battery percentage, and charging status when available.

This information is used to provide controller monitoring, status display, and troubleshooting details. It is not automatically sent to Synclab.

Overlay Usage

Batmon / Glimpse may request the Android overlay permission so it can show a small floating controller status overlay when the user enables that feature.

The app does not use Accessibility Service, does not record the screen, and does not control other apps. The overlay is intended only to show controller battery or status information.

Notifications and Foreground Service

The app may use notification permission and a foreground service for controller monitoring and overlay updates. The foreground service keeps monitoring visible to the user while the feature is active and can show controller status in a notification.

Report Problem

The Report Problem feature opens the user's email client with a prepared email. The user can review, edit, or cancel the email before sending it.

The prepared report may include text entered by the user, optional contact email, app version, Android version, device manufacturer/model, current monitoring state, overlay state, and redacted controller diagnostics. The app does not attach screen recordings or device log files in this phase.

Data Sharing

Synclab does not sell user data. Information is shared with Synclab only when the user chooses to send an email or otherwise contacts support. Email delivery is handled by the user's selected email provider.

Contact

For privacy questions, contact syncl4p+batmon.support@gmail.com.
